TCIMS — Place photos for Be@Mandaluyong "Explore"
=================================================

Drop a photo here named after the place (lowercase, words joined by hyphens, .jpg).
If a matching file exists, it shows on the card; otherwise a colored gradient shows.

Examples:
  San Felipe Neri Church          ->  san-felipe-neri-church.jpg
  Tatlong Bayani Statue           ->  tatlong-bayani-statue.jpg
  SM Megamall                     ->  sm-megamall.jpg

Use .jpg, recommended ~800x500px, landscape.
